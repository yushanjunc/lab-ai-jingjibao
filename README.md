# lab-ai-jingjibao

本仓库用于归档和发布多个独立 HTML 页面。  
页面内容主要涉及技术演示、工具页面或信息展示，不针对任何特定域名或网站。  
仓库名中的 "lab-ai-jingjibao" 为内部项目代号，无特殊含义。

## 目录说明

```
.
├── index.html          # 仓库首页 / 页面导航
├── pages/              # 存放各独立 HTML 页面
│   ├── page-01.html
│   ├── page-02.html
│   └── ...
├── assets/             # 共用的资源文件（CSS、JS、图片等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- **index.html**：可选的入口页面，用于快速浏览仓库内所有页面列表。  
- **pages/**：每个 HTML 文件均为一个独立的完整页面，可单独通过浏览器打开。  
- **assets/**：存放页面间共享的静态资源，便于统一维护。

## 页面归档说明

- 所有页面均为静态 HTML，不含后端依赖。  
- 页面之间相互独立，不共享数据或状态。  
- 文件名按内部约定命名，无特殊含义。  
- 页面内容可能随时间更新或替换，历史版本不保留。
